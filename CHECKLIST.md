## Common Issues & Solutions

### Issue: Mobile menu doesn't work
**Solution**: Make sure js/script.js is properly linked in your HTML files

### Issue: Styling looks wrong
**Solution**: Check that css/style.css path is correct in all HTML files

### Issue: Contact form doesn't work on GitHub Pages
**Note**: The form currently only has client-side validation. For production:
- Use a form service like Formspree, EmailJS, or Netlify Forms
- Or implement a backend server for form processing

### Issue: Images don't load
**Solution**: 
- Check image file paths (case-sensitive on Linux servers)
- Ensure images are in the correct folder
- Verify image file extensions match HTML code

### Issue: Website not updating on GitHub Pages
**Solution**:
- Wait 5-10 minutes after pushing changes
- Clear your browser cache (Ctrl+Shift+R or Cmd+Shift+R)
- Check if changes are in the correct branch
